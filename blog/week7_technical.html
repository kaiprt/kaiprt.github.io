Let's Talk SQL!

SQL

SQL stands for Structured Query Language. SQL is used to communicate with a database. According to ANSI (American National Standards Institute), it is the standard language for relational database management systems (SQLCourse.com). This week we will discuss SQL injection and why it is a problem in the SQL world!

SQL Injection

A SQL injection attack consists of insertion or "injection" of a SQL query via the input data from the client to the application (OWASP).


Why is this a problem?

Well if the attack is successful sensitive data can be accessed, altered, deleted, or updated within the database. This means bank balances can be changed, transactions voided, SSN's and credit card information stolen... get the picture? :) Fortunately, there are ways to protect your database from these vulnerabilities. Yay!
