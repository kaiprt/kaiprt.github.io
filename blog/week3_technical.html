Ruby- What exactly is looping? (Week 3)

In the Ruby programming language loops are used to execute the same block of code a specific number of times until a condition is met. There are a few basic loops:

  • While- a while loop executes a block of code as long as the statement in the block remains true.
  • Until- similiar to a while loop except it executes while the condition is false until true.
  • For- executes the block of code once for each element in an expression.

When choosing the right loop to use for your code it is important to assess what you want to accomplish in the loop. Lets say you want to right a block that puts a statement until a certain number. You would want the count to start from 0 and go to n. A while loop would be the best option for this instance because any number from 0 to n is less than or equal to n which a computer evaluates as true. If we used an until loop we would want the code to evaluate as false until true which doesn't quite make sense in our block because what we are evaluating is true. The for loop is like a hybrid between loops and iterators in Ruby. The for loop can go over values in a range or array.


We could possibly use a for loop in our block by writing:
for i in 1..10

end
