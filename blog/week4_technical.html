Ruby- Enumurable#cycle (Week 3)

The #cycle method used in Ruby is one of great convienience. #cycle calls a block of code for each element repeatedly forever or for a certain amount stated in parenthesis. The number stated in the paranthesis must be positive.
For example:
array = ["R", "u", "b", "y"]
print array.cycle { |i| puts i} # => R, u, b, y, R, u, b, y, R, u, b, y,.. FOREVER
print array.cycle(2) { |i| puts i} # => R, u, b, y, R, u, b, y. (Only cylces twice!)

As you can see #cycle is very beneficial in ruby because it allows you to run a block of code a set amount of times for each element. In my opinion I find cycle very similiar to .each and .times. #cycle seesm to work best in loops and for counting to a value.
