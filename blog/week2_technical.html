Why is JavaScript so popular in web development?

There are many reasons why JavaScript is very popular in web development today. From a programmers prospective, JavaScript is easy, fast, and powerful. JavaScript (or JS) is one of the easiest languages to work with. Most programmers agree that it is one of the easiest to learn with basic rules to follow and easy code debugging. JavaScript also features many libraries. Libraries are essentially "libraries" of pre-written JavaScript that allows easier development of JS based applications. This drastically speeds up application development. Since JS is based in your browser it takes the load off servers and provides offline functionality speeding up your website load time and making it easier on any given network. Instead of feeling removed from a website, JavaScript provides emersion for the user making the experience much more enjoyable.
